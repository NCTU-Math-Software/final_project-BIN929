# final_project-踩地雷
下載完所有檔案。執行main.m

# 游戲説明
### 開始需要輸入
* N: 矩陣大小最低下限是（9×9）最高上限是（16×16）
* Bomb_num： 炸彈的數量。下限是10顆 上限是(（N×N）/2)
### 游戲開始
#### 滑鼠左鍵
 * 是點開格子
 * 在有棋子的地方則不會點開
#### 滑鼠右鍵
 * 是在你覺得是炸彈的地方放棋子
 * 在有棋子的地方再點擊一次就會變回普通格子
#### Esc 
 * 强制停止當場游
#### 游戲結束後
 * 會再次詢問看看你有沒有想要在繼續玩如果想要的話按 1 就會繼續玩多一次。如果不想玩隨便輸入任何東西都可以
